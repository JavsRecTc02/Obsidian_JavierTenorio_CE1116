## 📚 Idea/Concepto 

Un prompt es el texto o instrucción que se proporciona a un modelo de lenguaje para guiar la generación de una respuesta. La calidad y claridad del prompt influyen directamente en la salida del modelo, ya que determina el contexto inicial que el sistema utiliza para predecir los siguientes tokens.
## 📌 Puntos Claves (Opcional)
- Entrada del usuario
- Control de generación
- Contexto inicial
