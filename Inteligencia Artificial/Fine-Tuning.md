## 📚 Idea/Concepto 

El fine-tuning es el proceso de ajustar un modelo previamente entrenado utilizando un conjunto de datos más específico para una tarea particular. En lugar de entrenar el modelo desde cero, se reutilizan los parámetros ya aprendidos y se adaptan para mejorar el rendimiento en dominios concretos.
## 📌 Puntos Claves (Opcional)
- Ajuste de modelos preentrenados
- Datos especializados
- Adaptación a tareas
