## 📚 Idea/Concepto 

Es el proceso de segmentar un texto en unidades llamadas tokens, generalmente como subpalabras, y luego mapear cada token a un identificador numérico mediante un vocabulario. Permite que los modelos procesen texto como datos numéricos, normalmente utilizando algoritmos de subpalabras como Byte Pair Encoding para manejar palabras desconocidas y equilibrar el tamaño del vocabulario con la longitud de las secuencias. Después de esto, los IDs son utilizados por la capa de embeddings para transformarse en vectores que capturan relaciones semánticas dentro del modelo.
